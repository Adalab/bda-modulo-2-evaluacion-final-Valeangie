gitEvaluación Final SQL - Base de Datos Sakila

Soy Valeria Mora y este es el repositorio con la resolución de los 24 ejercicios prácticos del Módulo 2, realizados sobre la base de datos Sakila.

Mi proceso
Como soy principiante en SQL, este proyecto ha sido mi primer contacto real con una base de datos compleja. He pasado de escribir consultas de una sola tabla a tener que conectar hasta cinco tablas distintas para obtener resultados específicos.#

Mis retos y aprendizajes
- Lo que más me costó: El ejercicio 23 fue el mayor desafío. Al principio intenté resolverlo solo con uniones, pero entender cómo usar una subconsulta con NOT IN para excluir datos me llevó tiempo. Fue un reto mental cambiar la lógica para buscar lo que NO existe en una lista.
- Mis retos con los JOIN: Al principio me resultaba difícil seguir el rastro de los ids a través de las tablas puente como film_actor o film_category. Tuve que dibujar los esquemas para entender cómo llegar desde el nombre de un actor hasta el nombre de una categoría.
- Orden y limpieza: He aprendido que comentar cada paso y hacer pequeñas pruebas antes de escribir la consulta definitiva es la única forma de no cometer errores y entender qué estoy haciendo en cada línea.

Gracias por revisar mi trabajo. Cada error que me ha dado la terminal durante este proceso ha sido una oportunidad para aprender algo nuevo.
